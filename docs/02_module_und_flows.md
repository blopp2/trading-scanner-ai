```markdown
# 📂 Modulübersicht & Datenfluss

## 📁 Projektstruktur

- `main.py`: Einstiegspunkt
- `vision_engine/`: Screenshot-Verarbeitung
- `core/`: Bewertung & Regeln
- `overlay/`: Visualisierung
- `analytics/`: Zeitverlauf & Alerts
- `docs/`: Doku

## 🔁 Datenfluss

1. Screenshot laden
2. GPT-4o oder OCR analysiert Bild
3. Tabelle wird extrahiert
4. Bewertungslogik berechnet Score
5. Ergebnis wird visuell angezeigt
```
